## Step 1: Send an Http Request

Define the data you need to verify

<CodeGroup>
```shell curl
curl https://app.inklink.com/api/request \
    --header "x-api-key: $secret_test_xxx" \
    --header "content-type: application/json" \
    --data \
'{
    "proof_request_description": "Verify Tesla ownership",
    "proof_title": "Vehicle Ownership Verification",
    "category": "OWNERSHIP",
    "return_url": "https://your-app.com/request-complete"
}'
```

```python Python
import requests

url = "https://app.inklink.com/api/request"

headers = {
    "x-api-key": "secret_test_xxx",
    "Content-Type": "application/json"
}

data = {
    "proof_request_description": "Verify Tesla ownership",
    "proof_title": "Vehicle Ownership Verification",
    "category": "OWNERSHIP",
    "return_url": "https://your-app.com/request-complete"
}

response = requests.post(url, headers=headers, json=data)

print(response.json())
```

```typescript TypeScript
const response = await fetch("https://app.inklink.com/api/request", {
  method: "POST",
  headers: {
    "x-api-key": "secret_test_xxx",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    proof_request_description: "Verify Tesla ownership",
    proof_title: "Vehicle Ownership Verification",
    category: "OWNERSHIP",
    return_url: "https://your-app.com/request-complete",
  }),
});

const data = await response.json();
console.log(data);
```

</CodeGroup>

#### Response

The response contains the URL given to users to complete the data verification process

```json JSON
{
  "request_id": "req_xxx",
  "request_url": "https://app.inklink.com/request/req_xxx"
}
```

### (Optional) Generate a Session Id

You can create a unique `session_id` to track when a specific user completes their request.

<CodeGroup>
```shell curl
curl https://app.inklink.com/api/request-session-id \
    --header "x-api-key: $secret_test_xxx" \
    --header "Content-Type: application/json" \
    --data \
'{
  "request_id": "req_xxx",
  "return_url": "https://your-app.com/request-complete?your-ref-id=xxx"
}'
```

```python Python
import requests

url = "https://app.inklink.com/api/request-session-id"

headers = {
    "x-api-key": "secret_test_xxx",
    "Content-Type": "application/json"
}

data = {
    "request_id": "req_xxx",
    "return_url": "https://your-app.com/request-complete?your-ref-id=xxx"
}

response = requests.post(url, headers=headers, json=data)

print(response.json())
```

```typescript TypeScript
const response = await fetch("https://app.inklink.com/api/request-session-id", {
  method: "POST",
  headers: {
    "x-api-key": "secret_test_xxx",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    request_id: "req_xxx",
    return_url: "https://your-app.com/request-complete?your-ref-id=xxx",
  }),
});

const data = await response.json();
console.log(data);
```

</CodeGroup>

#### Response

```json JSON
{
  "session_id": "sid_xxx",
  "request_url": "https://app.inklink.com/request/req_xxx?session_id=xxx"
}
```

## Step 2: Retrieve Request Results

<CodeGroup>

```shell curl
curl https://app.inklink.com/api/request-result/req_xxx \
    --header "x-api-key: $secret_test_xxx"
```

```python Python
import requests

url = "https://app.inklink.com/api/request-result/req_xxx"

headers = {
    "x-api-key": "secret_test_xxx",
}

response = requests.get(url, headers=headers)

print(response.json())
```

```typescript TypeScript
const response = await fetch(
  "https://app.inklink.com/api/request-result/req_xxx",
  {
    method: "GET",
    headers: {
      "x-api-key": "secret_test_xxx",
    },
  }
);

const data = await response.json();
console.log(data);
```

</CodeGroup>
