title: "Multi-Step Requests"
---

In addition to single proofs, InkLink also handles multi-step proofs in which a user proves multiple facts at the same time.

## Creating a Multi-Step Request

To create a multi-step request, use the `/request` endpoint and add the `request-steps` array to the JSON body:

<CodeGroup>
```bash curl highlight={9-11}
curl "https://app.inklink.com/api/request" \
  --header "x-api-key: $secret_test_xxx" \
  --header "Content-Type: application/json" \
  --data \
  '{
    "proof_title": "LinkedIn Account Information",
    "proof_request_description": "Capture all information about the user's experience, skills, education, account information on LinkedIn",
    "redirect_url": "https://www.linkedin.com/login",
    "request_steps": [
      ...
    ]
  }'
```

```python Python highlight={14-16}
import requests

url = "https://app.inklink.com/api/request"

headers = {
    "x-api-key": "secret_test_xxx",
    "Content-Type": "application/json"
}

data = {
    "proof_title": "LinkedIn Account Information",
    "proof_request_description": "Capture all information about the user's experience, skills, education, account information on LinkedIn",
    "redirect_url": "https://www.linkedin.com/login",
    "request_steps": [
        # ...
    ]
}

response = requests.post(url, headers=headers, json=data)

print(response.json())
```

```typescript Typescript highlight={12-14}
const response = await fetch("https://app.inklink.com/api/request", {
  method: "POST",
  headers: {
    "x-api-key": "secret_test_xxx",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    proof_title: "LinkedIn Account Information",
    proof_request_description:
      "Capture all information about the user's experience, skills, education, account information on LinkedIn",
    redirect_url: "https://www.linkedin.com/login",
    request_steps: [
      // ...
    ],
  }),
});

const data = await response.json();
console.log(data);
```

</CodeGroup>

## Request Steps

The `request_steps` array looks like this:

```json json
 "request_steps": [
    {
      "index": 1,
      "user_instruction": "Click on the capture button after you login",
      "trigger_signal": "https://www.linkedin.com/feed/",
      "redirect_url": null,
      "llm_instruction": null,
      "full_page": false,
      "claims": [
        {
          "field": "user_full_name",
          "required": true,
          "description": "The LinkedIn user full name"
        }
      ]
    },
    {
      "index": 2,
      //...
    },//...
  ]
```

| Value            | Description                                     |
| ---------------- | ----------------------------------------------- |
| index            | The index of the request step                   |
| user_instruction | The instructions given to the user at this step |
| trigger_signal   | The url that triggers the instruction popup     |
| redirect_url     | The starting url for the request step           |

{/* | redirect_message | message shown while redirecting             | */}
